 赛题目标
本次挑战赛的核心目标是检验 AI Agent 的“需求理解能力、自主决策能力、信息整合能力、结果输出能力”，最终开发出一款能够落地适配租房场景、真正解决用户痛点的 AI Agent，具体内容如下：
• 精准识别用户租房多元需求，包括但不限于租金预算、租房地段、户型要求、通勤半径、生活配套、房屋设施、租赁周期等，可应对模糊需求、隐含需求的解读；
• 自动完成房源信息的搜集、筛选、核验，排除虚假房源、不符合需求的房源，优先匹配高适配度房源；
• 能够对候选房源进行多维度分析，对比房源优缺点，结合用户需求给出针对性推
荐理由；
• 输出结果清晰、直观、实用，便于用户快速理解、决策，降低用户租房决策成本。
Agent 核心功能要求
参赛开发的 AI Agent 需具备以下核心功能，围绕“用户租房需求”实现全流程自动化处理：
1. 需求交互与解析：支持用户以自然语言（口语化、书面化均可）输入租房需求，能够精准提取核心需求点，对于模糊需求（如“想找一个通勤方便、性价比高的房子”），可主动追问补充关键信息，直至明确用户核心诉求；
2. 房源信息处理：具备自动搜集多平台房源信息并对比的能力，可基于模拟数据源能够对房源信息进行筛选、核验、分类（按适配度、租金、地段等维度）；
3. 多维度分析能力：能够对筛选后的候选房源，从通勤距离、租金性价比、生活配套（超市、医院、地铁等）、房屋设施、租赁风险（如隐性消费、房东资质）等维度进行分析，形成客观的优缺点评价；
4. 结果输出：最终输出最多 5 套高匹配度候选房源。
